SYMFONY SECRETS
###########################################
LIST
symfony console secrets:list --reveal


1 - SET
symfony console secrets:set PRIVATE_IDENTIFIANT --random
symfony console secrets:set PRIVATE_PASSWORD --random



#CHANGE
symfony console secrets:set PRIVATE_IDENTIFIANT --local
symfony console secrets:set PRIVATE_PASSWORD --local


 pour la production
symfony  console secrets:set REMEMBER_ME --prod


Dumper les variables d’environnement pour plus de rapidité en développement:
composer dump-env dev


REMOVE
 php bin/console secrets:remove REMEMBER_ME


 Générer vos secrets

symfony console secret:generate-keys
APP_ENV=dev symfony console secret:generate-keys


symfony console secret:set MARMITS_SECRET


symfony console secret:set APP_ENV=prod MARMITS_SECRET 


To improve performance (i.e. avoid decrypting secrets at runtime), you can decrypt your secrets during deployment to the "local" vault:
APP_RUNTIME_ENV=dev php bin/console secrets:decrypt-to-local --force


Lister les secrets :
symfony console secrets:list --reveal

Supprimer un secret :
symfony console secrets:remove REMEMBER_ME
