SYMFONY SECRETS
###########################################
LIST
symfony console secrets:list --reveal


1 - SET
symfony console secrets:set PRIVATE_PASSWORD --random
symfony console secrets:set PRIVATE_INDENTIFIANT --random


#CHANGE
symfony console secrets:set PRIVATE_PASSWORD --local
symfony console secrets:set PRIVATE_INDENTIFIANT --local

 pour la production
symfony  console secrets:set REMEMBER_ME --prod

Dumper les variables d’environnement :
composer dump-env dev


REMOVE
 php bin/console secrets:remove REMEMBER_ME


 Générer vos secrets

bin/console secret:generate-keys
APP_ENV=dev bin/console secret:generate-keys


symfony console secret:set MARMITS_SECRET


symfony console secret:set APP_ENV=prod MARMITS_SECRET 





Lister les secrets :
symfony console secrets:list --reveal

Supprimer un secret :
symfony console secrets:remove REMEMBER_ME
